# Spotify Statistics made with Svelte
View your most listened-to artists and tracks using the official Spotify API.

[View in browser](https://svelte-spotifystatistics.netlify.app/)

![Screenshot of the project](https://raw.githubusercontent.com/julianollivieira/svelte-spotifystatistics/main/screenshot.png)

# What i used for this project
- [Svelte (Cybernetically enhanced web apps)](https://svelte.dev/)
- [Water.css (A drop-in collection of CSS styles to make simple websites just a little bit nicer.)](https://watercss.kognise.dev/)
- [axios (Promise based HTTP client for the browser and node.js)](https://github.com/axios/axios/)
- [Official Spotify API](https://developer.spotify.com/documentation/web-api/)
